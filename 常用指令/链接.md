# 常用指令链接

## 实验

``` bash
# 创建一个文件 f1
echo "test file1" > f1

ln f1 f2          #创建f1的一个硬连接文件f2

ln -s f1 f3       #创建f1的一个符号连接文件f3

ls -li # -i参数显示文件的inode节点信息()

```

> 输出

```bash

# ls -li # -i参数显示文件的inode节点信息()
total 8
1573046 -rw-rw-r-- 2 anran anran 11 Dec 13 18:12 f1
1573046 -rw-rw-r-- 2 anran anran 11 Dec 13 18:12 f2
1573164 lrwxrwxrwx 1 anran anran  2 Dec 13 21:07 f3 -> f1
```

## 总结

依此您可以做一些相关的测试，可以得到以下全部结论：

1).删除符号连接f3,对f1,f2无影响；
2).删除硬连接f2，对f1,f3也无影响；
3).删除原文件f1，对硬连接f2没有影响，导致符号连接f3失效；
4).同时删除原文件f1,硬连接f2，整个文件会真正的被删除。
