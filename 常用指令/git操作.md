# git 常用操作

## 常用操作

## 修改默认分支名称

默认情况下,git 仓库的默认分支名称是 `master`,而在一些项目中,可能会使用 `main` 作为默认分支名称.

要修改默认分支名称,可以使用以下命令:

```bash
# 在本地仓库执行
git branch -m master main

# -m master main ：将本地的 master 分支重命名为 main
```

## 术语速查

Branch(分支)
> 分支是用来开发新功能或者修复 bug 的独立代码线,每个分支都是一个独立的版本

Commit(提交)
> 提交是对代码的一次修改,每次提交都有一个唯一的标识符(commit hash)

push(推送)
> 推送是将本地分支的提交上传到远程仓库,可以让其他开发者看到和使用你的修改

pull(拉取)
> 拉取是将远程仓库的提交下载到本地分支,可以更新本地代码和版本
