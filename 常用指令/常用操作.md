```bash
export PATH=$PATH:新环境变量的路径
```

```bash
# 备份源文件

```

## 安装常用开发工具

```bash
sudo apt update
# 一键装“编译必用”套装（gcc、g++、make、libc-dev、gdb 等）
sudo apt install -y build-essential
# 可选：装 32 位库、cmake、pkg-config、git 等常用搭档
sudo apt install -y gcc-multilib g++-multilib cmake pkg-config git
# 验证
gcc --version && g++ --version && make --version
```

## 删除 ssh 指纹
