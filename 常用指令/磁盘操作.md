# 磁盘操作指令

## mount 命令

挂载文件系统(操作u盘的时候可能会用到)

**注意**
    1. 挂载点必须是一个空目录
    2. 挂载点必须存在

```bash
sudo mount /dev/sdb1 /home/anran/data1
```

## umount 命令

卸载文件系统(操作u盘的时候可能会用到)

```bash
sudo umount /home/anran/data1
```

## df 命令

查看文件系统磁盘空间使用情况

```bash
# 查看文件系统磁盘空间使用情况
df -h
# 查看文件系统类型 (磁盘格式)
df -T
```

## 概念

### 文件系统

文件系统是操作系统用于管理和组织存储设备上的文件和目录的一种机制。它定义了文件和目录的存储位置、访问权限、元数据等信息。

#### 文件系统的种类有哪些

FAT VFAT NTFS EXT1 EXT2 EXT3 EXT4 HFS ...
vfat :bootloader (引导加载程序)
ext4 : 根目录
tmpfs : 内存文件系统

#### 什么是分区 

Liunx：按照功能来分区，每个分区严格存放文件 
    嵌入式文件系统可以分为四个分区：
    1. Bootloader   启动分区
    2. para         启动代码向内核传递参数的位置
    3. kernel       内核分区
    4. rootfs       文件系统结构
