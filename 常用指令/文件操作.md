
# 文件相关命令


[TOC]

## touch 命令

> 用来创建空文件或更新文件的时间戳

```bash
touch e1.sh
# 创建 e1.sh 文件
```

## file 命令

```bash
file e1.sh
# e1.sh: Bourne-Again shell script, ASCII text executable
```

## whereis 命令

> 用来寻找文件所在位置

```bash
whereis e1.sh
# e1.sh: /home/liunx/e1.sh
```

## scp 命令

> 用来在本地主机和远程主机之间复制文件

```bash
scp e1.sh liunx@192.168.1.100:/home/liunx/
# 将 e1.sh 文件复制到远程主机 192.168.1.100 的 /home/liunx/ 目录下
```



## head 命令 与 tail 命令

```bash
head -n 5 e1.sh
# 显示 e1.sh 文件的前 5 行

tail -n 5 e1.sh
# 显示 e1.sh 文件的最后 5 行
```

## more命令

```bash
more e1.sh
# 分页显示 e1.sh 文件的内容
```

## grep 命令

```bash
grep = e1.sh
# 检查 e1.sh 中包含等于号的行数
```

## md5sum 命令

```bash
md5sum e1.sh
# 显示 e1.sh 文件的 MD5 校验和
```

## du 命令

```bash
du e1.sh
# 显示 e1.sh 文件的大小（以字节为单位）

du -h e1.sh
# 显示 e1.sh 文件的大小（以人类可读的方式）
```
